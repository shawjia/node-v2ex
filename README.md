# v2ex

v2ex SDK, v2ex Cli

## Getting Started
Install the module with: `npm install v2ex`

```javascript
var v2ex = require('v2ex');
v2ex.home(function(err, items){});
```

Type `v2ex` in your terminal to use the built-in Cli.
